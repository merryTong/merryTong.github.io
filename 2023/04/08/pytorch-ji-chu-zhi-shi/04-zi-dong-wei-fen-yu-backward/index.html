Pytorch|自动微分和反向求导

基础知识 pytorch
Pytorch
发布日期:   2023-04-08

神经网络通常依赖反向传播求梯度来更新网络参数，求梯度过程通常是一件非常复杂而容易出错的事情。而深度学习框架可以帮助我们自动地完成这种求梯度运算。
Pytorch一般通过反向传播backward()方法实现这种求梯度计算。该方法求得的梯度将存在对应自变量张量的grad属性下。除此之外，也能够调用torch.autograd.grad()函数来实现求梯度计算。这就是Pytorch的自动微分机制。

求导数

利用backward方法求导数

backward()方法通常在一个标量张量上调用，该方法求得的梯度将存在对应自变量张量的grad属性下。
如果调用的张量非标量，则要传入一个和它同形状的gradient参数张量。相当于用该gradient参数张量与调用张量作向量点乘再求和，得到的标量结果再反向传播。

标量的反向传播

求$f(x) = a\times x^2 + b \times x + c$的导数。
$f(x)$的导数是$f’(x) = 2ax+b$。
当$x=0$时导数为-2。
当$x=2$时导数为2。
当$x=-2$时导数为-6。

x = torch.tensor(0.0, requires_grad = True) # x需要被求导
a = torch.tensor(1.0)
b = torch.tensor(-2.0)
c = torch.tensor(1.0)
y = a*torch.pow(x,2) + b*x + c 

print(y)
y.backward()
dy_dx = x.grad
print(dy_dx)

>>>Output:
tensor(1., grad_fn=<AddBackward0>)
tensor(-2.)

非标量的反向传播

求$f(x) = a\times x^2 + b \times x + c$的导数。

x = torch.tensor([[0.0,0.0],[1.0,2.0]],requires_grad = True) # x需要被求导
a = torch.tensor(1.0)
b = torch.tensor(-2.0)
c = torch.tensor(1.0)
y = a*torch.pow(x,2) + b*x + c 

gradient = torch.tensor([[1.0,1.0],[1.0,1.0]])

print("x:\n",x)
print("y:\n",y)
y.backward(gradient = gradient)
x_grad = x.grad
print("x_grad:\n",x_grad)

>>>Output:
x:
 tensor([[0., 0.],
        [1., 2.]], requires_grad=True)
y:
 tensor([[1., 1.],
        [0., 1.]], grad_fn=<AddBackward0>)
x_grad:
 tensor([[-2., -2.],
        [ 0.,  2.]])

用标量的反向传播实现非标量的反向传播

求$f(x) = a\times x^2 + b \times x + c$的导数。

x = torch.tensor([[0.0,0.0],[1.0,2.0]],requires_grad = True) # x需要被求导
a = torch.tensor(1.0)
b = torch.tensor(-2.0)
c = torch.tensor(1.0)
y = a*torch.pow(x,2) + b*x + c 

gradient = torch.tensor([[1.0,1.0],[1.0,1.0]])
z = torch.sum(y*gradient)

print("x:",x)
print("y:",y)
print(z)
z.backward()
x_grad = x.grad
print("x_grad:\n",x_grad)

>>>Output:
x: tensor([[0., 0.],
        [1., 2.]], requires_grad=True)
y: tensor([[1., 1.],
        [0., 1.]], grad_fn=<AddBackward0>)
tensor(3., grad_fn=<SumBackward0>)
x_grad:
 tensor([[-2., -2.],
        [ 0.,  2.]])

利用autograd.grad方法求导数

求$f(x) = a\times x^2 + b \times x + c$的导数。
$f(x)$的一阶导数是$f’(x) = 2ax+b$。
$f(x)$的二阶导数是$f’’(x)=2a$。
torch.autograd.grad方法中create_graph参数设置为 True 将允许创建更高阶的导数 。

x = torch.tensor(0.0,requires_grad = True) # x需要被求导
a = torch.tensor(1.0)
b = torch.tensor(-2.0)
c = torch.tensor(1.0)
y = a*torch.pow(x,2) + b*x + c

gradient = torch.autograd.grad(y,x,create_graph=True)
print(gradient)
dy_dx = gradient[0]
print(dy_dx.data)

# 求二阶导数
dy2_dx2 = torch.autograd.grad(dy_dx,x)[0] 
print(dy2_dx2.data)

>>>Output:
(tensor(-2., grad_fn=<AddBackward0>),)
tensor(-2.)
tensor(2.)

torch.autograd.grad允许同时对多个自变量求导数。如果有多个因变量，相当于把多个因变量的梯度结果求和。
$y_1 = x_1 \times x_2$
$y_2=x_1+x_2$
$\frac{\partial y_1}{\partial X} = \left[ \frac{\partial y_1}{\partial x_1} , \frac{\partial y_1}{\partial x_2} \right]= \left[x_2, x_1 \right]$
$\frac{\partial Y}{\partial X} =\left[ \left[ \frac{\partial y_1}{\partial x_1} , \frac{\partial y_1}{\partial x_2}, \right] \left[ \frac{\partial y_2}{\partial x_1} , \frac{\partial y_2}{\partial x_2}, \right] \right]= \begin{bmatrix}
x_2 & x_1\
1 & 1
\end{bmatrix}$

x1 = torch.tensor(1.0,requires_grad = True)
x2 = torch.tensor(2.0,requires_grad = True)

y1 = x1*x2
y2 = x1+x2

(dy1_dx1,dy1_dx2) = torch.autograd.grad(outputs=y1,inputs = [x1,x2],retain_graph = True)
print(dy1_dx1, dy1_dx2)

(dy12_dx1,dy12_dx2) = torch.autograd.grad(outputs=[y1,y2],inputs = [x1,x2])
print(dy12_dx1, dy12_dx2)

>>>Output:
tensor(2.) tensor(1.)
tensor(3.) tensor(2.)

利用自动微分和优化器求最小值

求$f(x) = a\times x^2 + b \times x + c$的最小值。

x = torch.tensor(0.0,requires_grad = True) # x需要被求导
a = torch.tensor(1.0)
b = torch.tensor(-2.0)
c = torch.tensor(1.0)

optimizer = torch.optim.SGD(params=[x],lr = 0.1)
for param in optimizer.state_dict():
    print(param, optimizer.state_dict()[param])

def f(x):
    result = a*torch.pow(x,2) + b*x + c 
    return(result)

for i in range(10):
    optimizer.zero_grad()
    y = f(x)
    print(y)
    y.backward()
    optimizer.step()
    print(optimizer.param_groups[0])

print("y=",f(x).data,";","x=",x.data)

反向传播的梯度累加

以下面的计算过程为例，进行反向传播计算。
$x_2 = x_1 \times w_1$
$y = x_2 \times w_2$
$L = Y-y$

x1 = torch.from_numpy(2*np.ones((2,2), dtype=np.float32))
x1.requires_grad_(True)
w1 = torch.from_numpy(5*np.ones((2,2), dtype=np.float32))
w1.requires_grad_(True)

print("x1 = ", x1)
print("w1 = ", w1)

>>>Output:
x1 =  tensor([[2., 2.],
        [2., 2.]], requires_grad=True)
w1 =  tensor([[5., 5.],
        [5., 5.]], requires_grad=True)
x2 = x1 * w1
w2 = torch.from_numpy(6*np.ones((2,2), dtype=np.float32))
w2.requires_grad_(True)

print("x2 = ", x2)
print("w2 = ", w2)

>>>Output:
x2 =  tensor([[10., 10.],
        [10., 10.]], grad_fn=<MulBackward0>)
w2 =  tensor([[6., 6.],
        [6., 6.]], requires_grad=True)
y = x2 * w2
Y = torch.from_numpy(10*np.ones((2,2), dtype=np.float32))
print("y = ", y)
print("Y = ", Y)

L = Y-y
print("Loss = ", L)

>>>Output:
y =  tensor([[60., 60.],
        [60., 60.]], grad_fn=<MulBackward0>)
Y =  tensor([[10., 10.],
        [10., 10.]])
Loss =  tensor([[-50., -50.],
        [-50., -50.]], grad_fn=<SubBackward0>)

$L = Y-y=Y-x_1 \times w_1 \times w2$
$\frac{\partial L}{\partial x_1} = \frac{\partial L}{\partial y} \times \frac{\partial y}{\partial x_1}=-w_1 \times w_2$
$\frac{\partial L}{\partial w_1} = \frac{\partial L}{\partial y} \times \frac{\partial y}{\partial w_1}=-x_1 \times w_2$
$\frac{\partial L}{\partial w_2} = \frac{\partial L}{\partial y} \times \frac{\partial y}{\partial w_2}=-x_1 \times w_1$

grad = torch.ones(2,2,dtype=torch.float)

L.backward(grad)

print("x1.grad = ", x1.grad)
print("w1.grad = ", w1.grad)
print("w2.grad = ", w2.grad)

>>>Output:
x1.grad =  tensor([[-30., -30.],
        [-30., -30.]])
w1.grad =  tensor([[-12., -12.],
        [-12., -12.]])
w2.grad =  tensor([[-10., -10.],
        [-10., -10.]])

张量的梯度是累加的，如果不对梯度清零，梯度会一直累加下去。
此时进行新的运算，并查看$x_1$的梯度。
$L_2 = x_1 \times x_1$
$\frac{\partial L_2}{\partial x_1} = 2 \times x_1$
经过第一轮运算后$x_1$的梯度是$\begin{bmatrix} -30 & -30 \ -30 & -30 \end{bmatrix}$，新的运算后计算得到$x_1$的梯度是$\begin{bmatrix} 4 & 4 & 4 & 4 \end{bmatrix}$，则累加后的$x_1$的最终梯度是$\begin{bmatrix} -26 & -26 \ -26 & -26 \end{bmatrix}$。

L2 = x1*x1
L2.backward(torch.ones(2,2, dtype=torch.float))
print("x1.grad = ", x1.grad)

>>>Output:
x1.grad =  tensor([[-26., -26.],
        [-26., -26.]])
