线性回归

基本原理

线性模型试图通过线性组合的方式对目标进行拟合或逼近。它的一般形式如$\hat y = f(x) = w_1x_1+w_2x_2+…+w_dx_d+b$

其中$ x_1,x_2,…,x_d$是我们的输入数据，$x_i$表示样本数据的某个特征，$ w_1,w_2,…,w_d $和$b$是线性回归模型的参数，将其改写为向量形式：

$\hat y = f(x) = w^T*x + b$

当建立了这样一个形如$\hat{y}=w*x+b$的模型之后，我们并不知道参数$w$和$b$的具体数值是多少，那么怎么确定参数$w$和$b$呢？首先考虑一种简单的情形：输入的样本数据只有一个特征。假设有一批数据${ (x_1,y_1),(x_2,y_2),…,(x_d,y_d) }$，已经知道数据$x_i$和对应的结果$y_i$，则可以定义线性回归模型的损失为：真实值$y$与预测值$\hat{y}$的差的平方和——MSE损失（Mean Square Error）。

$$\begin{aligned} L = \sum_{i=1}^{N} (y_i - \hat{y_i})^2 \end{aligned}$$

则线性回归模型的参数$w$和$b$可以通过下面的公式得到，当真实值$y$和预测值$\hat{y}$的损失达到最小时，此时线性回归模型的参数$w^*$和$b^*$就是我们要找的模型参数。

$$\begin{aligned} (w^{*}, b^{*}) = \arg \mathop{\min}\limits_{(w,b)} \sum_{i=1}^N (y_i-f(x_i))^2 \ = \arg \mathop{\min}\limits_{(w,b)} \sum_{i=1}^N (y_i-w*x_i - b)^2 \end{aligned}$$

由均方差损失（MSE）可知该损失函数是一个关于$w$和$b$的凸函数，当该函数对$w$和$b$的导数均为0时，该函数达到极小值，即可求得模型参数$w^*$和$b^*$。因此，将损失$L$对两个参数分别求导

$\begin{aligned} \frac{\partial L}{\partial w} = 2\sum_{i=1}^N (y_i - w*x_i - b)(-x_i)=2(w\sum_{i=1}^N x_i^2 - \sum_{i=1}^N(y_i -b)x_i) \end{aligned}$

$\begin{aligned} \frac{\partial L}{\partial b} = 2\sum_{i=1}^N (y_i - w*x_i - b)(-1)=2(Nb- \sum_{i=1}^N (y_i-wx_i)) \end{aligned}$

令导数$\frac{\partial L}{\partial b}=0$，则

$$\begin{aligned} Nb &=\sum_{i=1}^N (y_i-wx_i)\\ b &=\frac{1}{N}\sum_{i=1}^N (y_i-wx_i) \\ b^* &=\bar y - w \bar x \end{aligned}$$

令导数$\frac{\partial L}{\partial w}=0$，并将$b=\bar y - w \bar x $带入，则

$$\begin{aligned} w\sum_{i=1}^N x_i^2 &= \sum_{i=1}^N y_i x_i - \sum_{i=1}^N bx_i = \sum_{i=1}^N y_i x_i - \sum_{i=1}^N (\bar y-w\bar x)x_i \Rightarrow \\ w\sum_{i=1}^N x_i^2 &= \sum_{i=1}^N y_i x_i - \bar y \sum_{i=1}^N x_i + w\bar x \sum_{i=1}^N x_i \\ w &= \frac{\sum_{i=1}^N y_i x_i - \bar y \sum_{i=1}^N x_i }{\sum_{i=1}^N x_i^2 - \bar x \sum_{i=1}^N x_i} \end{aligned}$$

又有$\bar y \sum_{i=1}^N x_i = \frac{1}{N} \sum_{i=1}^N y_i \sum_{i=1}^N x_i = \bar x \sum_{i=1}^N y_i $，$\bar x \sum_{i=1}^N x_i = \frac{1}{N} \sum_{i=1}^N x_i \sum_{i=1}^N x_i=\frac{1}{N} (\sum_{i=1}^N x_i)^2$，将其代入上式：

$$\begin{aligned} w^* = \frac{\sum_{i=1}^N y_i(x_i - \bar x)}{\sum_{i=1}^N x_i^2 - \frac{1}{N}(\sum_{i=1}^N x_i)^2} \end{aligned}$$

推广到矩阵形式，若每个样本有d个特征，则线性回归模型的数学表达式为：

$$\hat{Y}=\begin{bmatrix} \hat{y}_1 \\ \hat{y}_2\\ \vdots\\ \hat{y}_N \end {bmatrix}=XW+b= \begin {bmatrix} x_1^1 & x_1^2 & \cdots & x_1^d\\ x_2^1 & x_2^2 & \cdots & x_2^d\\ \vdots & \vdots & \ddots & \vdots\\ x_N^1 & x_N^2 & \cdots & x_N^d \end {bmatrix} \begin {bmatrix} w_1 \\ w_2\\ \vdots\\ x_d \end {bmatrix} + b$$

为便于讨论将参数$W$和$b$合并为一个矩阵$\widehat W = (w; b)$，相应地样本数据$X$变为一个$N*(d+1)$大小的新矩阵$X$，如下式：

$$\hat{Y}=X \widehat W= \begin {bmatrix} x_1^1 & x_1^2 & \cdots & x_1^d & 1\\ x_2^1 & x_2^2 & \cdots & x_2^d & 1\\ \vdots & \vdots & \ddots & \vdots & \vdots \\ x_N^1 & x_N^2 & \cdots & x_N^d & 1 \end {bmatrix} \begin {bmatrix} w_1 \\ w_2\\ \vdots\\ x_d \\ b\end {bmatrix} $$

可以得出损失函数为如下形式

$L = (Y-X \widehat W)^T (Y-X \widehat W)$

则当损失$L$为最小时，可求得参数$\widehat W$为最优解，令损失$L$对参数$\widehat W$求导，得

$$\begin{aligned} \widehat W^* &= \arg \mathop{\min}\limits_{\widehat W} (Y-X \widehat W)^T (Y-X \widehat W) \Rightarrow \\ \frac{\partial L}{\partial \widehat W} &= \frac{\partial}{\partial \widehat W} \big[ Y^TY -Y^TX\widehat W - \widehat W^TX^TY + \widehat W^TX^T X \widehat W \big] \\ &=0-X^TY - X^TY + (X^TX+X^TX)\widehat W \\ &= 2X^T(X\widehat W-Y) \end{aligned}$$

令$\frac{\partial L}{\partial \widehat W} = 0$，则

$$\begin{aligned} 2X^T(X\widehat W-Y) &= 0 \Rightarrow X^TX \widehat W = X^T Y \\ \widehat W^* &= (X^TX)^{-1} X^TY\end{aligned}$$

代码实现

上面介绍了线性回归的基本原理，下面通过Python和Numpy来实现一个线性回归模型。这里使用波士顿房价预测的数据来举例，为了后面的可视化，自变量只用了房屋面积一个维度，即通过房屋面积去拟合房价。通过上面的公式推导，我们已经求出了线性回归模型的解析式，因此我们按照公式实现代码即可。
首先是数据读取，只取出房价和房屋面积的数据。

from sklearn.datasets import load_boston
import pandas as pd
import numpy as np

dataset = load_boston()

dataframe = pd.DataFrame(dataset["data"])
dataframe.columns = dataset["feature_names"]
dataframe["price"] = dataset["target"] 
# X房屋面积 y房价
X, y = dataframe["RM"].values, dataframe["price"] 
X, y = np.array(X), np.array(y)

# 训练集与测试集的简单划分
offset = int(X.shape[0] * 0.9)
X_train, y_train = X[:offset], y[:offset]
X_test, y_test = X[offset:], y[offset:]
X_train = X_train.reshape((-1,1))
X_test = X_test.reshape((-1,1))
y_train = y_train.reshape((-1,1))
y_test = y_test.reshape((-1,1))

print('X_train=', X_train.shape) # X_train= (455, 1)
print('X_test=', X_test.shape) # X_test= (51, 1)
print('y_train=', y_train.shape) # y_train= (455, 1)
print('y_test=', y_test.shape) # y_test= (51, 1)

按照公式实现参数计算的代码：

# 通过解析式计算参数
def linear_analysis(X, y):
    params_temp = np.linalg.inv(np.dot(X.T, X))
    params = np.dot(params_temp, np.dot(X.T, y))
    return params

# 线性模型的解析解
temp = np.ones((X_train.shape[0],1),dtype=np.float32)
X_new = np.concatenate((X_train, temp), axis=1)
params = linear_analysis(X_new, y_train)
params_analysis = {"w":params[0], "b":params[1]}
print(params_analysis)
# {'w': array([9.16394512]), 'b': array([-34.8343267])}

但是，我们看到参数的解析式中有求逆的操作，只有当$X^TX$是满秩矩阵或正定矩阵才会成立，当实际情况下$X^TX$不满秩则参数的解析解不唯一。

因此，这里我们使用剃度下降算法进行参数求解。首先定义模型的参数并初始化，参数$w$和$b$全部初始化为0。

def initialize_params(dims):
    w = np.zeros((dims, 1))
    b = 0
    return w, b

然后定义线性回归模型的计算、损失计算和求导计算。

def forward(X, w, b):
    # 模型公式
    y_hat = np.dot(X, w) + b 
    return y_hat

def cal_loss(y, y_hat):
    # 损失函数
    loss = np.mean((y_hat-y)**2)
    return loss

def cal_grad(X, y, y_hat):
    num_train = X.shape[0]
    # 参数的偏导
    dw = np.dot(X.T, (y_hat-y)) / num_train
    db = np.mean((y_hat-y))    
    return dw, db

这里使用梯度下降算法求解模型参数$w$和$b$。

def linear_train(X, y, learning_rate, epochs):
    w, b = initialize_params(X.shape[1])  
    loss_list = []  
    for i in range(1, epochs):        
        # 计算当前预测值、损失和参数偏导
        y_hat = forward(X, w, b)
        loss = cal_loss(y, y_hat) 
        loss_list.append(loss)      
        dw, db = cal_grad(X, y, y_hat)

        # 基于梯度下降的参数更新过程
        w += -learning_rate * dw
        b += -learning_rate * db   

        # 打印迭代次数和损失
        if i % 10000 == 0:
            print('epoch %d loss %f' % (i, loss)) 
               
        # 保存参数
        params = {            
            'w': w,            
            'b': b
        }        
        # 保存梯度
        grads = {            
            'dw': dw,            
            'db': db
        }    
    return loss_list, loss, params, grads

上面基本实现了一个线性回归的功能，为了检测我们实现的线性回归模型的有效性，仍然使用波士顿房价预测的数据进行测试，自变量只用了房屋面积一个维度，通过房屋面积去拟合房价。接着使用上述的训练数据训练模型，使用测试数据预测结果。

# 训练
loss_list, loss, params, grads = linear_train(X_train, y_train, 0.001, 100000)
# 预测结果并绘图
import matplotlib.pyplot as plt
f = X_test.dot(params['w']) + params['b']

plt.scatter(X_test, y_test)
plt.plot(X_test, f, color = 'darkorange')
plt.xlabel('X')
plt.ylabel('y')
plt.show()

最后，为了更好的评估模型，使用K折交叉验证来评估我们实现的线性回归模型，计算的指标是回归中常用的$R^2$。可以看到$R^2$的均值为168，结合上面测试集的散点图和拟合直线，说明回归的效果不是非常理想，这是因为但从房屋面积去拟合房价是有失偏颇的。

def predict(X, params):
    w = params['w']
    b = params['b']

    y_pred = np.dot(X, w) + b    
    return y_pred

def linear_cross_validation(data, k, randomize=True):   
    from sklearn.utils import shuffle
    if randomize:
        data = list(data)
        shuffle(data)

    slices = [data[i::k] for i in range(k)]      
    for i in range(k):
        validation = slices[i]
        train = list()
        for s in slices:
            if s is not validation:
                train.extend(s)

        train = np.array(train)
        validation = np.array(validation)            
        yield train, validation
        
# K折交叉验证
data = np.concatenate((X.reshape(-1,1), y.reshape(-1,1)), axis=1) 
print(data.shape)

r_square_list = []
for train, validation in linear_cross_validation(data, 5):
    X_train = train[:, :-1].reshape(-1,1)
    y_train = train[:, -1].reshape((-1, 1))
    X_valid = validation[:, :-1].reshape(-1,1)
    y_valid = validation[:, -1].reshape((-1, 1))

    loss_list, loss, params, grads = linear_train(X_train, y_train, 0.001, 100000)
    y_pred = predict(X_valid, params)
    r_square = np.mean((y_valid-y)**2)
    r_square_list.append(r_square)
print('five kold cross validation score is', r_square_list)
# five kold cross validation score is [162.41398318220567, 164.19161507455092, 178.86012366454037, 179.2822412241224, 159.51121355613822]
print('valid score is', np.mean(np.array(r_square_list)))
# valid score is 168.85183534031154
