深度学习中的图像操作

计算机视觉中图像增强是很实用的trick，下面介绍一下深度学习中常用的图像增强操作。

图像读取和保存

在Python中常用的图像处理的库有OpenCV和PIL，这里简单介绍一下这两个库的最基本用法。

opencv

cv2.imread()读取图像的代码示例如下，该方法返回得到的img数据类型是np.ndarray类型，使用img.shape可读取数组的维度，该维度的物理含义为(Height, Width, Channel)。

import cv2
img = cv2.imread("xxxxx.jpg")
# flags是可选的读入模式，如灰度图等，默认为None
img = cv2.imread("xxxxx.jpg", flages=cv2.IMREAD_GRAYSCALE)

img = cv2.imread("xxxxx.jpg")读取图像的通道顺序为BGR，即使图片是RGBA四通道，cv2.imread()方法仍然读取的是BGR三通道。可以通过使用cv2.split(img)分别得到图像img的BGR通道值。

b,g,r = cv2.split(img)

如果图像路径中存在中文、或是图像文件名含有中文，有时使用cv2.imread()是无法正常读取图像的，因此可以使用下面的方式读取和写入。

import cv2
import numpy as np
# 读取图像
img = cv2.imdecode(np.fromfile(img_path, dtype=np.uint8), cv2.IMREAD_COLOR)
# 保存图像
cv2.imencode(".jpg", img)[1].tofile(img_path)

PIL

使用PIL读取图像的代码示例如下，Image.open()返回的img数据类型是PIL.Image对象，不是普通的数组。使用img.size可读取图像的维度大小，该维度的物理含义为(Width, Height)，注意没有通道维度的大小。

from PIL import Image
# 读取图像
img = Image.open("xxx.jpg")
img = Image.open("xxx.jpg", mode=‘r’ ) # mode只能且默认是‘r’
# 保存图像
img.save("xxx.jpg")

Image.open()方法只是保持了图像被读取的状态，但是图像的真实数据并未被读取，因此如果需要操作图像每个元素，如输出某个像素的RGB值等，需要执行对象的load()方法读取数据。具体如下：

img = Image.open("xxx.jpg")
img = img.load()
print(img[0,0])
# result：(255, 201, 166)

Image.open()方法返回的默认彩色图像读取通道顺序为RGB，同时当图像格式为RGBA时，Image.open(‘xxx.jpg’)读取的格式为RGBA（其中A表示图像的alpha通道，即RGBA共四个通道），可以使用convert()函数转换为RGB

Image.open(x).convert('RGB')

opencv和PIL的格式转换

np.ndarray格式转换为PIL.Image格式

img = cv2.imread(path)
img_Image = Image.fromarray(np.uint8(img))

PIL.Image格式转换为np.ndarray格式

img = Image.open(path)
img_array = np.array(img)

torchvision中的图像预处理

torchvision是一个计算机视觉工具包，包含了常用的模型、数据集和图像预处理方法：

  • torchvision.transforms ：常用的图像预处理方法
  • torchvision.datasets：常用数据集的dataset实现，如MNIST，CIFAR-10等
  • torchvision.model：常用的模型预训练，AlexNet，VGG， ResNet等

基本操作

图像预处理的基本操作如下：

  1. transforms.Tensor()：将PIL图像或np.ndarray转换为torch.Tensor，并进行数值缩放。首先在数据格式上形如$HWC$的PIL图像和numpy数组转换为$CHW$，并将$[0,255]$的数值缩放到$[0,1]$，最后返回torch.FloatTensor的张量。

  2. transforms.Normalize()：对图像逐通道进行标准化操作，传入的参数有mean、std，分别表示均值和方差。输入是torch.Tensor，不支持直接对PIL图像进行操作。

  3. transforms.PILToTensor()：将PIL图像（$HWC$）转换为torch.Tensor（$CHW$），该操作不会进行数值缩放。

  4. transforms.ToPILImage()：将torch.Tensor形如（$CHW$）或np.ndarray形如（$HWC$）转换为PIL图像。该方法常用作张量的可视化。

裁剪

path = "lena_example/lena.jpg"
image = Image.open(path)
print(image.size) # (474, 474)

transforms.CenterCrop()：将PIL图像或是torch.Tensor进行中心裁剪。传入参数size是int整数或序列，表示裁剪的图像大小；如果输入数据的维度小于裁剪图像的大小，则会进行补0操作。

crop1 = transforms.CenterCrop(224)(image)
crop2 = transforms.CenterCrop((474,500))(image)
plt.subplot(1,2,1)
plt.imshow(crop1)
plt.subplot(1,2,2)
plt.imshow(crop2)crop1 = transforms.CenterCrop(224)(image)
crop2 = transforms.CenterCrop((474,500))(image)

transforms.RandomCrop()：将PIL图像或是torch.Tensor进行随机位置裁剪。传入参数size是int整数或序列，表示裁剪的图像大小；该方法的其余参数都是填充相关的，如下：

  • padding参数表示填充大小，当输入为a时表示上下左右均填充a个像素；当输入为(a, b)时，上下填充b个像素，左右填充a个像素；当输入为(a, b, c, d)时，表示左，上，右，下分别填充a, b, c, d个像素
  • pad_if_need参数表示如果裁剪图像小于size则进行填充。
  • padding_mode参数表示填充模式：”constant”、”edge”、”reflect”、”symmetric”，区别见下文Pad()。
  • fill参数表示填充的元素（当填充模式是constant时）
crop1 = transforms.RandomCrop(224)(image)
crop2 = transforms.RandomCrop((474,500), padding=50)(image)
crop3 = transforms.RandomCrop((474,500), pad_if_needed=True, padding_mode="edge")(image)
fig = plt.figure(figsize=(10, 6))
plt.subplot(1,3,1)
plt.imshow(crop1) 
plt.title("random")
plt.subplot(1,3,2)
plt.imshow(crop2)
plt.title("padding")
plt.subplot(1,3,3)
plt.imshow(crop3)
plt.title("padding_mode")

transforms.RandomResizedCrop()：将PIL图像或是torch.Tensor进行随机大小、长宽比的裁剪，并进行指定尺寸的缩放。第一个参数size表示裁剪图像的大小，第二个参数scale表示随机裁剪的面积比例，默认值是(0.08, 1.0)；第三个参数ratio表示长宽比，默认值是(3/4, 4/3)，第四个参数interpolation表示插值算法。

crop1 = transforms.RandomResizedCrop(224)(image)
crop2 = transforms.RandomResizedCrop((474,500))(image)
plt.subplot(1,2,1)
plt.imshow(crop1) 
plt.subplot(1,2,2)
plt.imshow(crop2)

transforms.FiveCrop()：图像的上下左右以及中心裁剪出尺寸为size的5张图片。返回值是一个元组，包含5张图像。

transforms.FiveCrop()：图像的上下左右以及中心裁剪出尺寸为size的5张图片，对这5张图片进行水平或者垂直镜像获得10张图片。返回值是一个元组，包含10张图像。

翻转

transforms.RandomHorizontalFlip()：依概率水平（左右）翻转图片，传入参数p表示概率。

transforms.RandomVerticalFlip()：依概率垂直（上下）翻转图片，传入参数p表示概率。

transforms.RandomRotation()：随机旋转图片，传入参数degrees表示旋转角度范围，当为a时，在（-a，a）之间选择旋转角度，当为(a, b)时，在(a, b)之间选择旋转角度。

图像变换

transforms.Pad()：对图片边缘进行填充。

  • 参数padding表示设置填充大小，当为a时，上下左右均填充a个像素，当为(a, b)时，上下填充b个像素，左右填充a个像素，当为(a, b, c, d)时，左，上，右，下分别填充a, b, c, d个像素。

  • 参数padding_mode：填充模式，有4种模式

    • constant：填充的像素值由fill参数设定
    • edge：像素值由图像边缘像素决定
    • reflect：镜像填充，最后一个像素不镜像，例如[1,2,3,4] → [3,2,1,2,3,4,3,2]
    • symmetric：镜像填充，最后一个像素也会镜像，例如[1,2,3,4] → [2,1,1,2,3,4,4,3]

  • 参数fill：设置填充的像素值。

crop1 = transforms.Pad(100)(image)
crop2 = transforms.Pad(100, padding_mode="edge")(image)
crop3 = transforms.Pad(100, padding_mode="reflect")(image)
fig = plt.figure(figsize=(10, 6))
plt.subplot(1,3,1)
plt.imshow(crop1) 
plt.title("pure")
plt.subplot(1,3,2)
plt.imshow(crop2)
plt.title("edge")
plt.subplot(1,3,3)
plt.imshow(crop3)
plt.title("reflect")

transforms.ColorJitter()：调整图像的亮度、对比度、饱和度和色相

  • brightness参数：亮度调整因子当为a时，从[max(0, 1-a), 1+a]中随机选择；当为(a, b)时，从[a, b]中随机选择。
  • contrast参数：对比度参数，同brightness
  • saturation参数：饱和度参数，同brightness
  • hue参数：色相参数，当为a时，从[-a, a]中选择参数，注： 0<= a <= 0.5；当为(a, b)时，从[a, b]中选择参数，注：-0.5 <= a <= b <= 0.5
img1 = transforms.ColorJitter(brightness=0.5, contrast=0, saturation=0, hue=0)(image)
img2 = transforms.ColorJitter(brightness=0, contrast=0.5, saturation=0, hue=0)(image)
img3 = transforms.ColorJitter(brightness=0, contrast=0, saturation=0.5, hue=0)(image)
img4 = transforms.ColorJitter(brightness=0, contrast=0, saturation=0, hue=0.5)(image)
fig = plt.figure(figsize=(8, 8))
plt.subplot(2,2,1)
plt.imshow(img1) 
plt.title("brightness")
plt.subplot(2,2,2)
plt.imshow(img2)
plt.title("contrast")
plt.subplot(2,2,3)
plt.imshow(img3) 
plt.title("saturation")
plt.subplot(2,2,4)
plt.imshow(img4)
plt.title("hue")

transforms.Grayscale()：灰度化图像，传入参数num_output_channels表示输出图像的通道数，只能为1或3。

transforms.RandomGrayscale()：随机灰度化图像，传入参数num_output_channels表示输出图像的通道数，只能为1或3，参数p表示转换为灰度图的概率。

transforms.RandomAffine()：对图像进行仿射变换，仿射变换是二维的线性变换，由五种基本原子变换构成，分别是旋转、平移、缩放、错切和翻转。该方法有如下参数，

  • degrees表示图像旋转角度
  • translate表示图像平移区间，如(a, b)，a设置宽，b设置高。图像在宽维度平移的区间为 -img_width * a < dx < img_width * a。
  • scale表示图像缩放比例（以面积为单位）
  • shear表示图像错切
  • fill_color表示填充颜色设置
img1 = transforms.RandomAffine(degrees=(-45,45))(image)
img2 = transforms.RandomAffine(degrees=(-45,45), translate=(0.3,0.8))(image)
img3 = transforms.RandomAffine(degrees=(-45,45),scale=(0.5,1))(image)
img4 = transforms.RandomAffine(degrees=(-45,45),shear=(-45,45))(image)
fig = plt.figure(figsize=(8, 8))
plt.subplot(2,2,1)
plt.imshow(img1) 
plt.title("rotate")
plt.subplot(2,2,2)
plt.imshow(img2)
plt.title("translate")
plt.subplot(2,2,3)
plt.imshow(img3) 
plt.title("scale")
plt.subplot(2,2,4)
plt.imshow(img4)
plt.title("shear")

transforms.RandomErasing()：对图像进行随机擦除。参数p表示概率值，执行随机擦除的概率；参数scale表示擦除区域的面积；参数ratio表示擦除区域的长宽比；参数value表示设置遮挡区域的像素值(R, G, B)或(Gray)。该操作不支持PIL图像。

image_tensor = transforms.PILToTensor()(image)
img1 = transforms.RandomErasing(p=1.0)(image_tensor)
img2 = transforms.RandomErasing(p=1.0, value=127)(image_tensor)
img3 = transforms.RandomErasing(p=1.0, value=(255,0,0))(image_tensor)
fig = plt.figure(figsize=(10, 6))
plt.subplot(1,3,1)
plt.imshow(transforms.ToPILImage()(img1)) 
plt.title("pure")
plt.subplot(1,3,2)
plt.imshow(transforms.ToPILImage()(img2))
plt.title("value=127")
plt.subplot(1,3,3)
plt.imshow(transforms.ToPILImage()(img3))
plt.title("rgb")

transforms的操作

transforms.Compose()：把一系列transforms方法集成到一起，并顺序执行transforms方法。

transforms.Compose([transforms1, transforms2, transforms3])

transforms.RandomChoice()：从一系列transforms方法中随机挑选一个。

transforms.RandomChoice([transforms1, transforms2, transforms3])

transforms.RandomApply()：依据概率执行一组transforms方法。

transforms.RandomApply([transforms1, transforms2, transforms3], p=0.5)

transforms.RandomOrder()：将一组transforms方法的操作顺序打乱。

transforms.RandomOrder([transforms1, transforms2, transforms3])

transforms.Lambda()：用户自定义transforms方法，传入参数lambda是一个匿名函数。比如FiveCrop()返回的元组数据，和GT可能不匹配时，可以如下操作。

trans = transforms.Compose([
    transforms.FiveCrop(size),
    transforms.Lambda(lambda crops: torch.stack([PILToTensor()(crop) for crop in crops]))
])
